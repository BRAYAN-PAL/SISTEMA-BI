# SISTEMA BI - Análisis de Congestión de Trámites

## Descripción

Sistema de Business Intelligence (BI) desarrollado para analizar la congestión de trámites administrativos mediante técnicas de análisis de datos, Machine Learning y visualización de información.

El proyecto permite identificar patrones, tiempos de atención, cuellos de botella y factores que influyen en la acumulación de trámites, facilitando la toma de decisiones basada en datos.

---

## Objetivo General

Analizar y predecir la congestión de trámites utilizando herramientas de Business Intelligence y Machine Learning para optimizar los procesos administrativos.

## Funcionalidades

* Registro y análisis de trámites.
* Procesamiento y limpieza de datos.
* Análisis de tiempos de atención.
* Identificación de cuellos de botella.
* Generación de indicadores de rendimiento.
* Predicción de niveles de congestión mediante Machine Learning.
* Visualización de métricas mediante Power BI.
